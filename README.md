# Fluidity
Fluidity is a Forge Utility Mod.

## Building
To Build Fluidity, you need to do these 2 commands below:
~~~shell
./gradlew setupDecompWorkspace
./gradlew build
~~~

## Installing
Put the jar into the mod folder, then run Forge
